## 📌주요기능
1.카테고리가 잘만들어 졌는가.

2.카테고리안에 음식이 잘 형성되어있는가.

3.다른숫자를 누르면 오류가 나지않는가.

4.잘못된 카테고리에 들어왔을시 뒤로가기 버튼으로 잘 넘어갈수있는가.

5.고른 음식의 정보가 잘 출력이되는가.
